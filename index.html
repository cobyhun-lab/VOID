📋 PATCH NOTES
v9.1 — CURRENT
⚔️ Stance System — choose a Loadout at run start. Each has a Red (attack) and Blue (control) weapon. Toggle mid-combat with F/Shift or the mobile Stance button. Red drains energy; Blue recharges it.
🎯 Four Loadouts — RAIDER (twin cannons + strafe burst), SENTINEL (pierce rifle + shield pulse), WARDEN (homing torpedo + siege mines), PHANTOM (sustained beam + gravity orbital).
💣 Warden reworked — Red stance now fires a tracking Proton Torpedo. Blue stance deploys a Siege Array: 3 proximity mines in formation, each triggering a 3-stage cluster detonation. Upgrades to 4 and 8 mines.
🛒 Shop reworked — weapons split into Red and Blue upgrade paths per loadout. All mines now cluster-detonate in 3 cascading stages.
🐛 Fixes — loadout selection now correctly applied on game start (audio wrapper was dropping the argument). Blue stance shop upgrades now route correctly. Tutorial progression fixed.
v8.3
🗄 Cloudflare D1 database — leaderboard migrated from JSONBin to D1 (SQLite at the edge). Proper concurrent writes, no race conditions, no external dependency.
🐛 End run bug fixed — game loop now fully stops when ending a run. No longer possible to return to menu and continue playing via the shop.
⚡ Performance improvements — particles capped at 120, damage floaters at 40, enemy bullets at 200 with off-screen culling. Canvas draw skipped entirely on menus and game over screen.
⚖️ Difficulty rebalance — enemy HP scaling reduced (1.15→1.10 per wave), bullet damage scaling reduced (8%→5% per wave), spawn counts trimmed, scrap drops increased for tier 1 & 2 enemies.
🛡 Screen boundary AI — all enemies now repelled from screen edges. No more units drifting off-screen on mobile.
⌨️ Bug report typing fixed — spacebar and other keys now work correctly in the report textarea.
🏆 Rank 0 bug fixed — leaderboard rank now always resolves correctly after score submission.
v8.1 — SECURITY & FIXES
🔐 Cloudflare Worker backend — API key removed from client. Leaderboard now proxied through a secure Worker with server-side validation.
🐛 Bug report system — pause menu → Report Bug. Captures wave, score, kills, weapon loadout, browser and version. Reports land in Discord instantly.
🏆 Leaderboard top 10 — only top 10 shown. Your rank displayed below if outside top 10. Version number shown on every entry.
🐛 END button fixed — confirm modal was hidden behind the shop overlay. Z-index corrected.
🐛 Crosshair crash fixed — stale shake offset variables removed from draw call.
v8.0 — CONTENT EXPANSION
🧠 Enemy AI overhaul — 6 distinct steering behaviours: Raiders flock, Bombers orbit, Gunships flank and switch sides, Interceptors make attack runs, Carriers hold position and deploy fighters.
👾 2 new enemy types — Interceptor (fast strafing attack runs, tier 2) and Carrier (slow, spawns TIE fighters every 5 seconds, tier 3).
💀 Boss every 10 waves — 3 escalating tiers. Each boss has 3 AI phases: orbit → charge → erratic. Fire patterns escalate from spread to rotating ring burst to fast aimed shots.
🌿 Beam Fork — projectile splits into a Y on first enemy hit [3×22 dmg].
💣 Mine Detonator — drops armed proximity mines behind you. 90 dmg AOE, 12 second lifespan.
⚓ Gravity Anchor — fires a projectile that plants a gravity well, pulling all nearby enemies in for 1.8 seconds.
🔥 Pulse Lance — fast pierce bolt that detonates an 80px AOE shockwave on hit [45 dmg + splash].
v7.4 — ECONOMY REBALANCE II
☕ Donation link added — subtle link on main menu and game over screen. buy coby a coffee.
💰 Economy tightened — buy limit 3/sector, all costs raised, gas drops reduced, util/proj cards shown reduced to 2.
🛡 4 new Ship Systems — Shield Capacitor, Bacta Infusion, Tibanna Magnet MK2, Hyperdrive Boost.
🔩 4 new Ordnance Mods — Chain Reaction, Void Lance, Scatter Matrix, Rapid Cycling.
🧹 Wave start cleanup — all bullets, mines and pulses cleared when jumping to a new sector.
v6.7 — SHOP & SYNERGY OVERHAUL
🔒 Shop lock redesigned — clear LOCK ITEM button on every card. Locked items highlight gold and survive rerolls.
🔫 Weapon synergies overhauled — 6 clean build paths: Rapidfire, Heavy, Scatter, Precision, Homing, Orbital, Pulse. No more circular suggestions.
💥 Shockwave Emitter — 3 concentric white rings expanding outward from your ship.
📵 Screen shake removed — smoother experience across all events.
v6.6 — EARLY GAME REBALANCE
⚡ Dual weapon slots from the start — second weapon slot unlocked immediately. No need to buy it.
🛒 More shop choices — Ship Systems and Ordnance Mods now show 3 cards per wave instead of 2.
⬡ Gas drops buffed — early enemies drop significantly more Tibanna.
